<h1>Менеджер подписок</h1>
<h3>Какую проблему мы решали? </h3> 
<p>В современном мире многие люди теряют огромные деньги на подписках, 
которые давно не используют, потому что следить за ними неудобно. Наше решение объединяет управление всеми подписками в одном окне — прозрачно, быстро и без лишней головной боли. Для организаторов конкурса это кейс не просто про техническую 
реализацию, а про реальную пользу и понимание современных проблем пользователей.</p>
<h3>Описание</h3>
<p>Web решение, включающее следующий функционал:</p>

1. регистрация пользователей
2. ручное добавление подписки
3. выбор подключаемой подписки из рекомендованных
4. аналитика трат за месяц / 3 месяца / 1 год
5. удаление и редактирование существующих подписок 
6. парсинг почты на наличие подписок, которых еще нет в системе
7. кроссплатформенный дизайн
8. Android-приложение для удобного использования

<h3>Содержимое проекта</h3>
В данном репозитории находяться все исходники нашей системы для ее локального запуска на любом устройстве, помимо этого
данный проект так же развернут на платформе railway.com и на него можно перейти по 
<a href='https://hackathon-production-ac65.up.railway.app/'>этой ссылке </a>

В репозитории помимо самого backend и frontend также располагается apk-файл для запуска сервиса 
с устройств под управлением ОС Android.



<h3>Установка необходимого ПО и зависимостей</h3>
<h4>Целевая система Windows 10 22H2</h4>

Cервис написан на микрофреймфорке Flask. Для его корректного запуска понадобиться любая IDLE поддерживающая python.
В нашем случае это был ```PyCharm```


1. Клонирование репозитория 
```git ```
2. Создание виртуального окружения
```python3 -m venv venv```
3. Активация виртуального окружения
```source venv/bin/activate```
4. Установка зависимостей
```pip3 install -r requirements.txt```
5. Настройка перед локальным запуском
Необходимо удалить все упоминания ```https://hackathon-production-ac65.up.railway.app/```  в файле ```index.html``` 
6. Запуск скрипта для демонстрации возможностей проекта
```python3 main.py --help```
